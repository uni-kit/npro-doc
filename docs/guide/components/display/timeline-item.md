## Props

| Prop name   | Description    | Type   | Values | Default  |
| ----------- | -------------- | ------ | ------ | -------- |
| bgType      | 背景主题       | string | -      | ''       |
| border      | 边框主题       | string | -      | ''       |
| radius      | 圆角主题       | string | -      | ''       |
| centerStyle | 中间线外层样式 | string | -      | ''       |
| lineBgType  | 中间线背景主题 | string | -      | 'border' |
| lineStyle   | 中间线样式     | string | -      | ''       |
| boxStyle    | 外层样式       | string | -      | ''       |
